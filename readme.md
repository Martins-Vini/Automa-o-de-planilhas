# Leitura de dados de planilhas e automação

**Proposta:**

Trata-se de um programa que lê e retorna dados e valores das células de qualquer página de planilha de qualquer arquivo que desejar. Podendo ter inovações, edições da planilha e mescla de planilha de um arquivo com outro.

Também é possível criar planilhas a partir de um arquivo qualquer, no exemplo do programa, uso um arquivo de texto com nome de  e algumas informações, separados por vírgula para ditar a formatação da planilha.

Para rodar: 

``` bash

python app.py

```

E edite as informações necessárias conforme sua necessidade.
